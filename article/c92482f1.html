软件推荐

发表于 2022-08-09 更新于 2023-01-03 阅读次数： 本文字数： 904 阅读时长 ≈ 3 分钟

推荐好用的软件

因为平台不同我会标注PC：Win/Mac/Linux，移动：Android/iOS

目前分类，基础软件、记录写作、学习提升、编程、浏览器插件、小程序。。。

基础软件

geek（Win）

快速彻底的一款卸载软件，极轻量，用了之后再没用过控制面板。

火绒安全（PC）

如其名，安全相关的，无广告，无强制捆绑，简单清爽。

腾讯柠檬（Mac）

Mac平台下的一款清理软件，免费好用。

Bandzip（Win/Mac）

轻量全能的解压缩软件，超级好用。与盛传的7zip相比界面更好看些，尽管有广告（说实话这个我真的无感，推出广告好长时间我都没发现，正常使用完全不用在意这个）。

Chrome（Win/Mac）

这个更不用多说，唯一遗憾可能是国内无法使用谷歌的一些服务吧。

Edge（Win/Mac）

Edge与Chrome的差距真的不大，现在都是Chromium内核，插件生态，各种兼容。之前一篇对比文章，点这。

夸克（Android/iOS）

还是很好用的手机浏览器。

NeatDownloadManager（Win/Mac）

超级推荐的一款下载器，相比于IDM，免费。配合浏览器插件就可以起飞了。

记录写作

语雀

推荐：PC网页+移动APP

之前用的也是Typora，总所周知它开始收费了，更重要的是它无法满足我需要的云（数据同步，多端使用）需要，就转向语雀了😊。感兴趣的看这篇MD编辑器的烦恼。

图床、素材更新ing

学习提升

这方面的软件、网站太多了，耳熟能详的就不提了。

简讯（Adnroid/iOS）

轻阅读，打发空闲时间（3、5分钟那样的），全方面的知识。

编程

VSCode（PC）

免费，而且几乎是全能的。

IDEA（Win/Mac）

jetbrains出品正在全方面占领市场，靠的就是硬硬的真实力。

Navicat（Win/Mac）

个人最喜欢的数据库管理工具，就是好用。

MobaXterm（Win）

SSH客户端、网络工具，可惜只有Win版，要是有Mac版，我相信会爆。

浏览器插件

都是Chrome和Edge所支持的，可能所有Chromium都支持，我也不知道😂

AdBlock

广告拦截器

Tampermonkey

知名的油猴，我玩的不是很六，自己发掘吧🤧

NeatDownloadManager Extension

配合NeatDownloadManager起飞，飞。

Octotree - GitHub code tree

如果你有网页查看GitHub代码跳转的困惑，那用它就完事了。

JSONVue

浏览器JSON格式化显示，当然用Postman更专业😒，但有时确实是需要用一下😏。

划词翻译

本来作为谷歌翻译的替代品，但越用越喜欢。多源翻译可选择，多种翻译方式，设置自定义化程度很高。

小程序

总结

好软件的标准是什么，我常在思考。

差生文具多。在开始做什么事之前，我总想着找到最好的工具，帮助我快速高效地达成目标，然而这也导致我花了大把时间在搜寻工具上，却忘了我还站在原地呢。所以啊，不要在搜寻工具上迷失了。

谈起好的软件，我个人认为首先就是免费（doge）。哈哈哈哈哈，不是。

好的软件肯定是用户友好的，符合使用习惯的，学习成本低的，易上手的。

关于轻量和全面，有人希望它们各司其职，各干各的，术业有专攻；有人希望能全面发展，一个顶几个，能干各种活。关于这个就仁者见仁智者见智了。
